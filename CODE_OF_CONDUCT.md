# Code of Conduct

This project adopts the **Contributor Covenant 2.1** as its code of conduct.

Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

To report an incident, open
a [GitHub Private Security Advisory](https://github.com/BrunoMartinsCorrea/gtk-cross-platform/security/advisories/new)
or e-mail the maintainer at **brunomsft@gmail.com**. All reports are reviewed within 7 days.

---

> For interactions in GNOME channels (Matrix, GNOME GitLab, mailing lists), the
> [GNOME Code of Conduct](https://conduct.gnome.org/) also applies.
